Instructions for Icons

For a production app, you would need to replace the placeholder icons with actual icon files:

  • icon-192x192.png (192×192 pixels)
  • icon-512x512.png (512×512 pixels)
  • maskable-icon.png (Adaptive icon with safe zone for different shapes)
  • record-icon.png (96×96 pixels for shortcuts)

You can use this HTML page as a template for your icon design.
